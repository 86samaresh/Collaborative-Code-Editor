# Real-Time Collaborative Code Editor

A VS Code-inspired collaborative code editor that enables multiple users to edit code together in real time.

## 🚀 Features (Planned)

- 📂 File Explorer
- 📝 Monaco Code Editor
- ⚡ Real-time collaboration using Socket.IO
- 💬 Integrated Chat
- 🖥️ Terminal & Console
- 📑 Multiple File Tabs
- 👥 Multi-user Room Support
- ▶️ Code Execution
- 💾 Project Persistence

## 🛠️ Tech Stack

### Frontend
- React
- Monaco Editor
- CSS

### Backend
- Node.js
- Express
- Socket.IO

### Database
- MongoDB
